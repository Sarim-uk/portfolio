Virtual Knowledge Distillation

Project Overview

An implementation of the research paper "A Virtual Knowledge Distillation via Conditional GAN" by Sihwan Kim. This project uses a Conditional GAN framework to facilitate knowledge transfer between a large teacher model (ResNet-50) and a smaller student model (ResNet-18), enabling efficient deployment of deep learning models without compromising accuracy.

Key Features

  • Conditional GAN-based knowledge distillation architecture
  • Synthesizes feature representations to bridge the gap between teacher and student models
  • Reduces dependency on large datasets through feature generation
  • Achieves model compression while maintaining comparable performance
  • Complete implementation in Jupyter Notebook with training and evaluation modules

Technical Challenges & Solutions

The main challenge was designing an effective generator that could synthesize high-quality feature representations for the student model. We implemented a carefully tuned Conditional GAN architecture that successfully captured the complex distribution of features from the teacher model. This approach enabled knowledge transfer without direct access to the original dataset, making it an efficient solution for model compression and deployment in resource-constrained environments.

Technologies Used

PyTorch torchvision Python Jupyter Notebook Deep Learning GANs ResNet numpy matplotlib

Unified Gesture Recognition & Fingertip Detection

Project Overview

An implementation of a unified convolutional neural network (CNN) algorithm for simultaneous hand gesture recognition and fingertip detection in egocentric vision. This project is based on a research paper published in Pattern Recognition journal, which presents a novel approach where a single network predicts both finger class probabilities for classification and fingertip positions for regression in one evaluation.

Key Features

  • Robust real-time hand detection using YOLO (You Only Look Once)
  • Unified network that performs both classification and regression tasks
  • Ensemble approach for fingertip position detection from a fully convolutional network
  • Support for 8 different hand gesture classes
  • Real-time processing capability with webcam input
  • Comprehensive visualization tools for model outputs

Technical Challenges & Solutions

The main challenge was developing a single model that could simultaneously handle the classification task (gesture recognition) and the regression task (fingertip detection) with high accuracy. The solution involved a two-stage approach: first detecting the hand using YOLO, then passing the cropped hand image to the unified CNN model. Instead of directly regressing fingertip positions from fully connected layers, we implemented an ensemble approach from a fully convolutional network, achieving better localization accuracy while maintaining real-time performance.

Technologies Used

TensorFlow OpenCV Python YOLO CNN Computer Vision ImgAug Keras NumPy

Titanic Data Analysis

Project Overview

A comprehensive data science project that analyzes the historical Titanic dataset to identify the most significant factors that influenced passenger survival. The analysis employs various data preprocessing and feature selection techniques to extract meaningful insights from this well-known dataset.

Key Features

  • Systematic data preprocessing pipeline including handling missing values and feature encoding
  • Multiple feature selection techniques including correlation analysis, statistical methods, and PCA
  • In-depth analysis of factors affecting survival rates (gender, class, fare, age)
  • Visualization of data patterns and correlations using matplotlib and seaborn
  • Statistical validation of findings using ANOVA F-value and Chi-Square tests
  • Well-documented Jupyter notebooks with clear explanations of methodologies

Technical Challenges & Solutions

The main challenge was dealing with the dataset's missing values and converting categorical variables to a format suitable for analysis. I implemented a robust preprocessing pipeline that handles missing age values through mean imputation and missing embarkation points through mode imputation. For categorical variables, I applied one-hot encoding for embarkation ports and label encoding for gender. The feature selection process required implementing and comparing multiple techniques to identify the truly significant factors, revealing that gender, passenger class, and fare were the most important predictors of survival.

Technologies Used

Python Jupyter Notebook Pandas NumPy Matplotlib Seaborn Scikit-learn Statistical Analysis Feature Engineering
